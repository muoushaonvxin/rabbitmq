
有的时候为了将rabbitmq当中的队列消息清空，但是又不想使用web界面的方式来做，这时可以使用代码的方式来完成

```php
<?php
error_reporting(E_ALL);
ini_set('display_errors', '1');
ini_set('date.timezone','Asia/Shanghai'); 
set_time_limit(0);
$conn = new AMQPConnection(array(
    'host' => "192.168.1.1",
    'port' => "5672",
    'login' => "guest",
    'password' => "guest",
    'vhost' => '/'
));

// 创建连接和channel
if (!$conn->connect()) {
    die("Cannot connect to the broker");
}

$e_name = 'zhenshiceshi';
$q_name = 'Queue111113';
$k_route = '111113';

$channel = new AMQPChannel($conn);
$ex = new AMQPExchange($channel);
$ex->setName($e_name);
$ex->setType(AMQP_EX_TYPE_FANOUT);

// 创建接受绑定反馈的队列Queue
$queue = new AMQPQueue($channel);
$queue->setName($q_name);
$queue->setFlags(AMQP_DURABLE);
$queue->declareQueue();

// 绑定交换机与队列，并指定路由键
$queue->bind($e_name, $k_route);
$queue->purge();

?>
```

在到web界面上去查看队列的信息发现消息被清空了

<br>

#### 使用代码清空多个队列当中的消息
```php

<?php
error_reporting(E_ALL);
ini_set('display_errors', '1');
ini_set('date.timezone','Asia/Shanghai'); 
set_time_limit(0);
$conn = new AMQPConnection(array(
    'host' => "192.168.1.234",
    'port' => "5672",
    'login' => "guest",
    'password' => "guest",
    'vhost' => '/'
));

// 创建连接和channel
if (!$conn->connect()) {
    die("Cannot connect to the broker");
}

// 创建接受绑定反馈的队列Queue
$channel = new AMQPChannel($conn);
$queue = new AMQPQueue($channel);

$queue_array = Array("Q111111","Queue111113","test01","Queue111111","Queue111112");
foreach($queue_array as $k){
    $queue->setName($k);
    $queue->setFlags(AMQP_DURABLE);
    $queue->declareQueue();
    $queue->purge();
}
?>

```

运行该代码将队列消息清空
```shell
[root@zhangyz ~]# php purge.php 
清除队列 Q111111 消息成功
清除队列 Queue111113 消息成功
清除队列 test01 消息成功
清除队列 Queue111111 消息成功
清除队列 Queue111112 消息成功
```
